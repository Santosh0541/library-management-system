# 📚 Library Management System

A web-based library management application to manage books, members, issue/return transactions, and fine calculation.

## 🔗 Live Demo
👉 [https://library-manag-s.netlify.app/](https://library-manag-s.netlify.app/)

## 📸 Features

- ✅ Add and manage books with genre, ISBN, and multiple copies
- ✅ Issue books to members with date tracking
- ✅ Return books with automatic fine calculation (₹2/day after 14 days)
- ✅ Full transaction history
- ✅ Dashboard with total books, available, issued counts
- ✅ Search books by title or author
- ✅ Data persists using localStorage
